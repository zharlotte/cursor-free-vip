## 🛠️ 如果还是看不到 Actions

### 检查仓库权限：
1. 进入仓库设置：`Settings` → `Actions` → `General`
2. 确保选择：**"Allow all actions and reusable workflows"**

### 检查分支：
- Actions 只在主分支（`main`）上显示
- 确认你查看的是主分支

### 强制刷新：
- 按 `Ctrl+F5` (Windows) 或 `Cmd+R` (Mac) 刷新页面
